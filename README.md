# NOTES 
## __STRINGS__
* Continuous string combinations formula `n*(n+1)/2`.

## __DYNAMIC PROGRAMMING__
* Use static allocation for dp grid. eg `static int dp[5001][5001];`
* General formula for n-steps problem `dp[i] = dp[i-1] + dp[i-2] + dp[i-3] + ...... + dp[i-n];`.
